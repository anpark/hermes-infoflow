MesloLGS NF, MesloLGM NF, and MesloLGL NF web fonts for RemoteCC terminal rendering.

Source TTF files were fetched from:
https://github.com/romkatv/powerlevel10k-media

The checked-in files are full-glyph WOFF2 conversions used by browsers that do
not have MesloLGS NF, MesloLGM NF, MesloLGL NF, or another Nerd Font installed locally.
